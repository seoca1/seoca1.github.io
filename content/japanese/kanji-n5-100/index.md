# 한자 N5: 필수 한자 100자 (漢字 N5 100 Essential Kanji)

> 🇰🇷 **한국어 요약 (Korean Summary)**
> - **핵심:** JLPT N5 급 한자 100자. 일본어 학습의 세 번째 문자 (히라가나 → 가타가나 → 한자).
> - **한국어 화자 최대 장점:** 한자 중 **70%가 한국 한자어와 뜻 공유** (意味 단어). 한자 뜻(meaning)은 쉽게 외움. **읽기(음독)만 별도 학습** 필요.
> - **음독 2종:** 音読み (음독 = On'yomi, 중국 음) + 訓読み (훈독 = Kun'yomi, 일본 고유 음).
> - **학습 순서:** 숫자 → 시간 → 방향 → 인체 → 자연 → 동사 → 형용사 → 명사
> - **다음 단계:** N4 한자 300자 + 단어 결합

## 한자란?

**한자 (漢字, kanji)**는 중국에서 온 문자로, 일본어에서 **의미 단위**로 사용됩니다. 각 한자는 1개 이상의 뜻(意味)과 1-2개의 전형적 읽기(음독)를 가집니다. 히라가나/가타가나가 **음절 문자**인 것과 달리, 한자는 **의미 문자(表意文字)** — 글자 자체가 의미를 나타냅니다.

**한국어 학습자에게 한자는 양날의 검:**

| 측면 | 장점 | 단점 |
|:---|:---|:---|
| **의미 (meaning)** | 70%가 한국 한자어와 의미 공유 (예: 食 = 한국 한자 食 = 먹을 식) | 새로운 음독 학습 필요 |
| **읽기 (reading)** | 같은 한자가 한국 한자에 있음 | 중국어식 음이 일본어와 다름 (예: 食 = 한국 식, 일본 ショク) |
| **쓰기** | 획순 비슷 | 일부 한자는 획순 다름 (예: 飛) |
| **학습 효율** | 한자 뜻은 외울 필요 없음 (한국어에 이미 있음) | 음독만 새로 외우면 됨 |

> **결론:** 한국어 화자는 한자 학습이 **영어/중국어 화자보다 훨씬 쉬움**. 뜻은 자동 인식, 음독만 학습.

## 음독 vs 훈독 (음독 2종)

### 음독 (音読み, On'yomi)
- **중국에서 유래한 읽기**
- 주로 **복합어 (熟語, jukugo)** 에서 사용
- 한국어 한자음과 비슷 (일본식 중국어)
- 예:
  - 学校 (ガッコウ, gakkō) = 학교
  - 日本 (ニホン, nihon) = 일본
  - 勉強 (ベンキョウ, benkyō) = 공부

### 훈독 (訓読み, Kun'yomi)
- **일본 고유 읽기**
- 단독 사용 또는 **억리가나 (送り仮名, okurigana)** 와 함께
- 한국어 한자음과 다름
- 예:
  - 食べる (たべる, taberu) = 먹다
  - 行く (いく, iku) = 가다

### 다중 음독 한자

많은 한자가 여러 음독을 가집니다. 예:
- **行**: コウ (음독) / いく (훈독, 동사) / おこなう (훈독, "수행하다")
- **生**: セイ / ショウ (음독) / いきる / うまれる / はえる / き (훈독)

## 한국어 학습자가 자주 하는 실수 5가지

1. **한국 한자음 ≠ 일본 음독** — 食은 한국 "식", 일본 "ショク". 의미만 공유.
2. **같은 한자, 다른 음독** — 学校 reads がっこう (gakkō), 하지만 다른 단어에서 学 might be がく or まな.
3. **획순 다름** — 한국 한자와 비슷하지만 일부 한자 (예: 飛)는 일본에서 획순이 다릅니다. 앱으로 연습.
4. **단순 ≠ 쉬움** — 1획 한자 一는 의미 단순, 10획 한자 家는 더 연습 필요. 획수 ≠ 난이도.
5. **한국어 단어 그대로 안 됨** — 공부 = 勉強 (べんきょう, benkyō), NOT 공부 (gongbu). 일본어 음독 별도.

## 8단계 학습법

### Level 1 — 숫자 (10자)
1-10 양쪽 음독 외우기.

### Level 2 — 시간·방향 (21자)
시간, 요일, 위치 한자 마스터.

### Level 3 — 신체·가족 (12자)
일상 어휘 기초.

### Level 4 — 자연 (8자)
자연 세계.

### Level 5 — 동사 (15자)
행동 동사 — 문장 구성의 핵심.

### Level 6 — 형용사 (10자)
묘사 표현.

### Level 7 — 명사 (14자)
일반 사물 + 추상 개념.

### Level 8 — 다중 음독 한자 (10자)
行 / 生 / 入 등 여러 음독 가진 한자.
