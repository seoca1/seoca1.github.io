# 병음 기초 (Pinyin Basics, 拼音)

> 🇰🇷 **한국어 요약 (Korean Summary)**
> - **핵심:** 중국어의 로마자 표기 시스템 (병음, pinyin) — 한자를 모르고도 중국어 발음을 적을 수 있음.
> - **구성:** 성모 (초성, initials 21개) + 운모 (중성/종성, finals 38개) + 4성조 (성조 4종).
> - **성조:** 1성 (고평, ā) / 2성 (상승, á) / 3성 (저하+상승, ǎ) / 4성 (하강, à) + 경성 (light).
> - **한국어 학습자 팁:** 한국어 한자 발음과 중국어 병음이 **다름**. 한자음(한글) ≠ 병음 (예: 中 = 한국 한자 "중" ≠ 중국병음 "zhōng").
> - **다음 단계:** [Numbers](../../lessons/numbers-1-to-100/) → [Greetings](../greetings/) → 단어/문장 구성

## 병음이란?

**병음 (拼音, pīnyīn)**은 중국어 발음을 **로마자 알파벳**으로 표기하는 시스템입니다. 1958년 중국 본토에서 표준화되었으며, 현재 중국 본토와 싱가포르에서 사용됩니다 (대만은 주음부호/병음 둘 다 사용).

한자를 모르는 사람도 병음을 알면 중국어 발음을 읽고 쓸 수 있습니다 — 중국어 학습의 **첫 단계**.

**한국어 학습자를 위한 중요 사실:**

> 한국 한자음 ≠ 중국 병음
>
> 예: 中 (한자 의미 = 가운데/중심)
> - 한국 한자음: "중" (jung)
> - 중국 병음: "zhōng" (zhong1)
> - 일본 음독: チュウ (chū)

한자 뜻은 공유하지만 **발음은 완전히 다름**. 중국어를 배울 때는 한국 한자음에 의존하면 안 됩니다.

## 성조 4종 (四声)

중국어는 **성조 언어 (聲調語言, tone language)** — 같은 발음도 성조에 따라 의미가 바뀝니다.

| 성조 | 부호 | 설명 | 예시 |
|:---:|:---:|:---|:---|
| **1성** | ā | 고평 (고음 유지) | 妈 (mā) = 어머니 |
| **2성** | á | 상승 (중→고) | 麻 (má) = 마 (삼씨) |
| **3성** | ǎ | 저하+상승 (저→저저→상승) | 马 (mǎ) = 말 |
| **4성** | à | 하강 (고→저) | 骂 (mà) = 꾸짖다 |
| **경성** | a | 성조 없음 (가벼움) | 吗 (ma) = ~인가? |

### 성조 쌍 — 의미를 바꾸는 예시

같은 발음 mā/má/mǎ/mà 가 의미가 모두 다름:

| 병음 | 한자 | 의미 |
|:---|:---|:---|
| mā | 妈 | 어머니 |
| má | 麻 | 마 (삼씨) |
| mǎ | 马 | 말 (동물) |
| mà | 骂 | 꾸짖다 |

> **경고:** 성조 틀리면 의미 통째로 바뀜. 처음부터 정확히 발음 연습 필수.

### 한국어 학습자 성조 매핑

| 중국어 성조 | 한국어 발음 매핑 | 학습 난이도 |
|:---:|:---|:---:|
| 1성 (고평) | 한국어 평성 비슷 | 쉬움 |
| 2성 (상승) | 한국어 억양 상승 (예: "아→?" 질문) | 중간 |
| 3성 (저하+상승) | 한국어 억양 저하 (예: "저요") | 어려움 |
| 4성 (하강) | 한국어 단단한 톤 (예: "맞아!" 단언) | 중간 |

## 성모 (Initials, 聲母) — 초성 21개

성모는 **자음으로 시작하는** 발음. 한국어 자음과 비슷하지만 다른 발음도 있음.

### 기본 성모 21개

| 성모 | 발음 가이드 | 예시 한자 |
|:---:|:---|:---|
| b | 한국어 "ㅂ"과 비슷 (무성, 약한 파열음) | 爸 (bà, 아버지) |
| p | 한국어 "ㅍ" (유성 파열음) | 怕 (pà, 두려워하다) |
| m | 한국어 "ㅁ" | 妈 (mā, 어머니) |
| f | 한국어 "ㅍ" (치경 마찰음) | 飞 (fēi, 날다) |
| d | 한국어 "ㄷ" (무성) | 大 (dà, 크다) |
| t | 한국어 "ㅌ" | 他 (tā, 그) |
| n | 한국어 "ㄴ" | 你 (nǐ, 너) |
| l | 한국어 "ㄹ" (치경 설측음) | 了 (le, ~했음) |
| g | 한국어 "ㄱ" (무성) | 哥 (gē, 형) |
| k | 한국어 "ㅋ" | 可 (kě, ~할 수 있다) |
| h | 한국어 "ㅎ" (후두 마찰음) | 好 (hǎo, 좋다) |
| j | 한국어 "ㅈ" (치경 마찰음, 더 날카로움) | 家 (jiā, 집) |
| q | 한국어 "ㅊ" (유기, 더 강한 기식) | 七 (qī, 7) |
| x | 한국어 "ㅅ" (치경 마찰음) | 西 (xī, 서쪽) |
| zh | 한국어 "ㅈ" (후치경 파열음, 말아올림) | 中 (zhōng, 가운데) |
| ch | 한국어 "ㅊ" (후치경 파열음) | 吃 (chī, 먹다) |
| sh | 한국어 "ㅅ" (후치경 마찰음) | 是 (shì, ~이다) |
| r | 한국어 "ㄹ" (치경 설측 근음) | 人 (rén, 사람) |
| z | 한국어 "ㅈ" (치경 파열음) | 在 (zài, ~에 있다) |
| c | 한국어 "ㅊ" (치경 파열음) | 次 (cì, 횟수) |
| s | 한국어 "ㅅ" (치경 마찰음) | 三 (sān, 3) |

> **한국어 학습자 주의:**
> - **j, q, x** vs **zh, ch, sh** — 한국어에는 둘 다 없어 구분 어려움. zh/ch/sh는 혀를 더 뒤로 말아올림.
> - **r** — 한국어 ㄹ과 다름. 혀를 말아서 발음.
> - **z, c, s** vs **zh, ch, sh** — z/c/s는 혀를 앞에만, zh/ch/sh는 혀를 말아올림.

운모는 **모음** 또는 **모음+자음(n/ng)** 으로 끝나는 발음. 단독으로 쓰일 수도 있음.

## 성조 변화 규칙 (Tone Sandhi)

중국어는 단어 안에서 성조가 변하는 규칙이 있습니다:

### 3성 + 3성 → 3성 + 2성

| 원형 | 변화 | 의미 |
|:---|:---|:---|
| 你好 (nǐ hǎo) | ní hǎo | 안녕 |
| 小姐 (xiǎo jiě) | xiǎo jié | 아가씨 |

> **규칙:** 두 번째 3성이 2성으로 바뀜. 첫 번째는 그대로.

### "一 (yī, 1)" 변화

| 위치 | 변화 | 예시 |
|:---|:---|:---|
| 단독 | yī | 一个 (yī gè, 한 개) |
| 4성 + 一 | yí | 十一 (shí yí, 11) |
| 1성 + 一 | yì | 一只 (yì zhī, 한 마리) |

### "不 (bù)" 변화

| 위치 | 변화 | 예시 |
|:---|:---|:---|
| 4성 앞 | bú | 不对 (bú duì, 틀리다) |
| 그 외 | bù | 不好 (bù hǎo, 안 좋다) |

## 한국어 학습자가 자주 하는 실수 5가지

1. **한국 한자음 ≠ 병음** — 의존 금지. 한국 한자음 (예: "중") ≠ 중국 병음 (예: zhōng). 처음부터 병음으로 발음 훈련.
2. **성조 무시** — 한국어와 달리 중국어는 성조가 의미를 바꿈. mā ≠ mǎ ≠ mà. 처음부터 성조 정확히.
3. **j/q/x와 zh/ch/sh 혼동** — 한국어에 없는 두 음소 계열. j/q/x는 혀 앞, zh/ch/sh는 혀 말아올림. 처음부터 구분 연습.
4. **ü 발음 어려움** — 한국어 "위"와 다름. 한국어 "ㅜ" + "ㅟ" 사이. 鱼 (yú, 물고기), 女 (nǚ, 여자) 등.
5. **성조 변화 규칙 무시** — 你好 (nǐ hǎo) → ní hǎo (3성 → 2성). 처음엔 규칙 몰라도 OK, 점차 적용.

## 5단계 학습법

### Level 1 — 성조 + 기본 모음 (6개 모음)
ā á ǎ à + a o e i u ü 정확히 발음.

### Level 2 — 성모 21개
b/p/m/f/d/t/n/l/g/k/h/j/q/x/zh/ch/sh/r/z/c/s.

### Level 3 — 기본 음절 읽기
성모 + 운모 결합. 각 음절 정확히.

### Level 4 — 단어/문장 (1-2주)
실전 단어 + 성조 변화 규칙 적용.

### Level 5 — 한자 도입
병음 → 한자 연결.
