# 🇰🇷 Korean Lessons

Korean (한국어) language learning resources for **English-speaking learners**.

## 📚 Browse by Type

- **Grammar** — sentence order (SOV), verb conjugation, particles
- **Lessons** — Hangul alphabet, pronunciation, greetings, basic phrases, particles
- **Vocabulary** — numbers, family, food

## 🌏 Why Korean?

- **77M+ native speakers** (Korea + diaspora)
- **Logical writing system** — Hangul was designed in 1446 by King Sejong to be learnable in days
- **K-content wave** — K-pop, K-drama, K-movies driving global interest
- **Career opportunities** — Samsung, LG, Hyundai, plus Korea's growing tech sector

## 🚀 Quick Start

1. **Day 1:** Learn Hangul (24 basic letters) — most learners finish in 1 week
2. **Week 2:** Pronunciation rules (받침, 연음, 경음화)
3. **Week 3:** Basic phrases + particles (은/는, 이/가, 을/를)
4. **Month 2:** Sentence order + verb conjugation basics
5. **Month 3+:** Themed vocabulary + conversation practice
