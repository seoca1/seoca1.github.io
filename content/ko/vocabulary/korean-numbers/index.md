> 🇰🇷 **운영자용 한국어 요약**
> 1. 한국어 숫자는 **두 가지**: **Sino-Korean** (한자 어원, 공식) + **Native Korean** (고유어)
> 2. Sino-Korean (일/이/삼/사/오/육/칠/팔/구/십): 돈, 날짜, 주소, 전화번호, 분, 초
> 3. Native Korean (하나/둘/셋/넷/다섯/여섯/일곱/여덟/아홉/열): 나이, 개월, 사물 개수
> 4. **만 (萬) = 10,000** — 10000 단위 (한국은 만 단위 절단). 1억 = 100,000,000
> 5. 단위: 십(10), 백(100), 천(1000), 만(10000), 억(100000000), 조(1000000000000)

# Korean Numbers: Two Systems

Korean uses **two different counting systems** depending on what's being counted. This is unique among world languages and a common stumbling block.

## 1. The Two Systems

| Number | Sino-Korean | Native Korean |
|:---:|:---:|:---:|
| 1 | 일 (il) | 하나 (hana) |
| 2 | 이 (i) | 둘 (dul) |
| 3 | 삼 (sam) | 셋 (set) |
| 4 | 사 (sa) | 넷 (net) |
| 5 | 오 (o) | 다섯 (daseot) |
| 6 | 육 (yuk) | 여섯 (yeoseot) |
| 7 | 칠 (chil) | 일곱 (ilgop) |
| 8 | 팔 (pal) | 여덟 (yeodeol) |
| 9 | 구 (gu) | 아홉 (ahop) |
| 10 | 십 (sip) | 열 (yeol) |
| 100 | 백 (baek) | — |
| 1,000 | 천 (cheon) | — |
| 10,000 | 만 (man) | — |
| 100,000,000 | 억 (eok) | — |

> Note: 100+ exists **only in Sino-Korean**. Native Korean uses a different word for higher numbers (열둘 = 12, 서른 = 30, 마흔 = 40).

## 2. When to Use Which System

### Use **Sino-Korean** for:
| Context | Example |
|:---|:---|
| Money | 일천 원 (₩1,000) |
| Dates | 2026년 칠월 십오일 (2026-07-15) |
| Phone numbers | 일공삼-일이삼사-오육칠팔 |
| Minutes/Seconds | 오 분 삼십 초 (5 min 30 sec) |
| Addresses | 서울특별시 강남구 테헤란로 칠십삼 길 |
| Floors | 삼 층 (3rd floor) |
| Math, percentages, measurements | 50% = 오십 퍼센트 |

### Use **Native Korean** for:
| Context | Example |
|:---|:---|
| Age (alone) | 스물다섯 살 (25 years old) |
| Months of age | 다섯 달 (5 months old) |
| Counting objects | 사과 세 개 (3 apples) |
| Hours (whole) | 한 시 (1 o'clock) |
| People | 두 명 (2 people) |
| Repetition | 한 번 (one time) |

> 💡 Time is mixed: **Sino-Korean for minutes/seconds** (오 분), but **Native Korean for full hours** (한 시, 두 시, 세 시).

## 3. Compound Numbers (Sino-Korean)

Build larger numbers **multiplicatively**:

| Number | Sino-Korean | Breakdown |
|:---:|:---|:---|
| 11 | 십일 | 10 + 1 |
| 23 | 이십삼 | 2×10 + 3 |
| 50 | 오십 | 5×10 |
| 100 | 백 | 1×100 |
| 234 | 이백삼십사 | 2×100 + 3×10 + 4 |
| 1,000 | 천 | 1×1,000 |
| 5,000 | 오천 | 5×1,000 |
| 12,345 | 만 이천삼백사십오 | 10,000 + 2,000 + 300 + 40 + 5 |

## 4. Compound Numbers (Native Korean, up to 99)

For ages and hours:

| 10–19 | 20–29 | 30–39 | 40–49 | 50–59 |
|:---|:---|:---|:---|:---|
| 열하나 (11) | 스물하나 (21) | 서른하나 (31) | 마흔하나 (41) | 쉰하나 (51) |
| 열둘 (12) | 스물둘 (22) | 서른둘 (32) | 마흔둘 (42) | 쉰둘 (52) |
| 열셋 (13) | 스물셋 (23) | 서른셋 (33) | 마흔셋 (43) | 쉰셋 (53) |

Special forms: 스물 (20), 서른 (30), 마흔 (40), 쉰 (50)

## 5. Native Counters (조사)

Korean uses **counters** with objects. The counter changes based on the noun type:

| Counter | For | Example |
|:---|:---|:---|
| **명** | people (formal) | 손님 세 **명** |
| **분** | people (honorific) | 선생님 두 **분** |
| **개** | most objects | 사과 다섯 **개** |
| **마리** | animals | 고양이 두 **마리** |
| **잔** | cups (drinks) | 커피 세 **잔** |
| **대** | vehicles / machines | 차 한 **대** |
| **권** | books | 책 네 **권** |
| **장** | flat things (paper) | 사진 두 **장** |
| **병** | bottles | 물 한 **병** |
| **그릇** | bowls | 밥 두 **그릇** |
| **살** | years of age | 스무 **살** (20 years old) |
| **시** | hours | 세 **시** (3 o'clock) |
| **일** | days | 다섯 **일** (5 days) |
| **개월** | months | 세 **개월** (3 months) |
| **년** | years | 다섯 **년** (5 years) |
| **원** | won (currency) | 일만 원 (₩10,000) |

> 💡 **Native Korean numbers** are used with most counters (개, 마리, 잔, etc.).
> 💡 **Sino-Korean numbers** are used with time-related counters (시, 분, 일, 개월, 년) and monetary (원).

## 6. Big Numbers — Man (만) = 10,000

Korean **truncates at 만 (10,000)**, not thousand:

| Number | Korean | Notes |
|:---:|:---|:---|
| 1,000 | 천 (cheon) | 1 thousand |
| 10,000 | 만 (man) | 10 thousand |
| 100,000 | 십만 (sip-man) | 100 thousand |
| 1,000,000 | 백만 (baek-man) | 1 million (not 천천!) |
| 10,000,000 | 천만 (cheon-man) | 10 million |
| 100,000,000 | 억 (eok) | 100 million |
| 1,000,000,000 | 십억 (sip-eok) | 1 billion |

> ⚠️ A common mistake: "일백만" (100만) instead of "백만" (백 + 만).
> ⚠️ $1 million = 백만 원, NOT 천천 원.

## 7. Practical Examples

| Context | Korean |
|:---|:---|
| ₩5,000 | 오천 원 |
| Phone: 010-1234-5678 | 공일공 일이삼사 오육칠팔 |
| Age 28 | 스물여덟 살 (or 스물여덟) |
| 3:30 PM | 오후 세 시 삼십 분 |
| 5 apples | 사과 다섯 개 |
| 2 dogs | 개 두 마리 |
| 7 people | 사람 일곱 명 |
| 12 months | 십이 개월 (or 일 년) |
| January 1st | 일월 일일 |
| 2026 | 이천이십육 년 |

## 8. Practical Tips

- ☑️ **Practice numbers 1-10 in BOTH systems** — they sound different
- ☑️ **Learn counters with objects** — 사과 세 개, 개 두 마리
- ☑️ **Remember 만 = 10,000** — the #1 source of mistakes
- ☑️ **Use Native Korean for ages** — 스물다섯 살
- ☑️ **Sino-Korean for money and time** — 만 원, 삼 분
