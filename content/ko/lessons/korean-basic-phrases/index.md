# Korean Basic Phrases

## 1. The 8 Essential Survival Phrases

These eight phrases cover most beginner situations:

| # | Korean | Pronunciation | English |
|:---:|:---|:---|:---|
| 1 | **네** | ne | Yes |
| 2 | **아니요** | anio | No |
| 3 | **안녕하세요?** | annyeong-haseyo | Hello |
| 4 | **감사합니다** | gamsahamnida | Thank you (formal) |
| 5 | **죄송합니다** | joesonghamnida | I'm sorry (formal) |
| 6 | **주세요** | juseyo | Please give me / I'd like... |
| 7 | **잘 모르겠어요** | jal moreugesseoyo | I don't know well |
| 8 | **화장실 어디예요?** | hwajangsil eodiyeyo? | Where is the bathroom? |

## 2. Polite Requests

| Korean | Pronunciation | English |
|:---|:---|:---|
| **주세요** | juseyo | Please give me... |
| **~주세요** | ~ juseyo | ...please (appended to noun) |
| **~해 주세요** | ~ hae juseyo | Please do... |
| **~주세요요** | (informal) | ...please (casual) |
| **~할 수 있어요?** | ~hal su isseoyo? | Can you...? |

### Examples
- **물 주세요** = Water, please
- **메뉴 주세요** = Menu, please
- **도와주세요** = Help, please
- **천천히 말해 주세요** = Please speak slowly

## 3. Want / Like / Need

| Pattern | Korean | English |
|:---|:---|:---|
| **~고 싶어요** | ~go sipeoyo | I want to... |
| **~이/가 좋아요** | ~i/ga joayo | I like... |
| **~이/가 싫어요** | ~i/ga sireoyo | I don't like... |
| **~가 필요해요** | ~ga piryohaeyo | I need... |
| **~이/가 없어요** | ~i/ga eopseoyo | I don't have... |

### Examples
- **한국 음식을 먹고 싶어요** = I want to eat Korean food
- **커피가 좋아요** = I like coffee
- **도와주세요, 도움이 필요해요** = Help, I need help
- **돈이 없어요** = I don't have money

## 4. Location & Direction

| Korean | Pronunciation | English |
|:---|:---|:---|
| **여기** | yeogi | Here |
| **거기** | geogi | There (near you) |
| **저기** | jeogi | Over there (far) |
| **오른쪽** | oreunjjok | Right (direction) |
| **왼쪽** | oenjjok | Left (direction) |
| **위** | wi | Up / Top |
| **아래** | arae | Down / Bottom |
| **안** | an | Inside |
| **밖** | bak | Outside |
| **앞** | ap | Front |
| **뒤** | dwi | Back |

Asking directions:
- **~가/이 어디예요?** = Where is ~?
- **~에 어떻게 가요?** = How do I get to ~?
- **~까지 멀어요?** = Is ~ far?

## 5. Time Expressions

| Korean | Pronunciation | English |
|:---|:---|:---|
| **지금** | jigeum | Now |
| **아까** | akka | A while ago |
| **나중에** | najunge | Later |
| **오늘** | oneul | Today |
| **내일** | naeil | Tomorrow |
| **어제** | eoje | Yesterday |
| **모레** | more | Day after tomorrow |
| **그저께** | geujeokke | Day before yesterday |

### Clock time
- **한 시** = 1 o'clock (1시)
- **두 시 삼십 분** = 2:30 (2시 30분)
- **오전** = AM
- **오후** = PM

> Note: Korean uses Sino-Korean numbers for time: 1=일/한, 2=이/두, 3=삼/석, 4=사/넉, 5=오, 6=육/여섯...

## 6. Money & Shopping

| Korean | Pronunciation | English |
|:---|:---|:---|
| **얼마예요?** | eolmayeyo? | How much is it? |
| **비싸요** | bissayo | It's expensive |
| **싸요** | ssayo | It's cheap |
| **깎아주세요** | kkakkajuseyo | Please give a discount |
| **카드로 계산할게요** | kadeu-ro gyesan-halgeyo | I'll pay by card |
| **현금으로 계산할게요** | hyongeum-euro gyesan-halgeyo | I'll pay with cash |

## 7. Emergency Phrases

| Korean | English |
|:---|:---|
| **도와주세요!** | Help! |
| **경찰을 불러주세요** | Please call the police |
| **의사가 필요해요** | I need a doctor |
| **병원에 가야 해요** | I need to go to the hospital |
| **길을 잃었어요** | I'm lost |
| **영어 할 줄 아는 사람 있어요?** | Is there someone who speaks English? |
| **다시 한 번 말해 주세요** | Please say it again |
| **천천히 말해 주세요** | Please speak slowly |
| **한국어 못해요** | I can't speak Korean |

## 8. Common Daily Phrases

| Korean | English |
|:---|:---|
| **잘 자요** | Good night |
| **좋은 꿈 꿔요** | Sweet dreams |
| **수고하셨어요** | Good work / You worked hard |
| **잘 먹겠습니다** | I will eat well (before meals) |
| **잘 먹었습니다** | I ate well (after meals) |
| **화이팅!** | Fighting! (you can do it!) |

## 9. Practical Tips

- ☑️ **Add ~요 to most phrases** — keeps you polite
- ☑️ **주세요 is your best friend** — works for most requests
- ☑️ **Carry your hotel's Korean name card** — easier for taxis
- ☑️ **Numbers differ for time vs counting** — Sino-Korean (일/이/삼) for time, native (하나/둘/셋) for counting
- ☑️ **Smile and bow slightly** when speaking — it goes a long way
