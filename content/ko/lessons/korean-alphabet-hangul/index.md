# Korean Alphabet: Hangul (한글)

## 1. The 14 Basic Consonants

Korean has **14 basic consonants**. Each is named after the shape it depicts:

| Letter | Name | Sound (IPA) | Mnemonic |
|:---:|:---|:---:|:---|
| ㄱ | 기역 (giyeok) | /k/, /g/ | Looks like a person holding a gun (k = gun) |
| ㄴ | 니은 (nieun) | /n/ | Looks like a person sitting on the ground |
| ㄷ | 디귿 (digeut) | /d/, /t/ | Like a C-shape opening to the right |
| ㄹ | 리을 (rieul) | /r/, /l/ | Looks like a tongue |
| ㅁ | 미음 (mieum) | /m/ | Looks like a mouth (ㅁ = square mouth) |
| ㅂ | 비읍 (bieup) | /b/, /p/ | Like a bucket |
| ㅅ | 시옷 (siot) | /s/ | Like a tooth (s = sharp) |
| ㅇ | 이응 (ieung) | /ŋ/ silent at start | Like a circle (ng) |
| ㅈ | 지읒 (jieut) | /dʒ/, /tʃ/ | Like a j-shape |
| ㅊ | 치읓 (chieut) | /tʃʰ/ | Aspirated version of ㅈ |
| ㅋ | 키읔 (kieuk) | /kʰ/ | Aspirated ㄱ |
| ㅌ | 티읕 (tieut) | /tʰ/ | Aspirated ㄷ |
| ㅍ | 피읖 (pieup) | /pʰ/ | Aspirated ㅂ |
| ㅎ | 히읗 (hieut) | /h/ | Like a person exhaling |

> 💡 The "aspirated" consonants (ㅋ, ㅌ, ㅍ, ㅊ) carry a strong puff of air — like English "p" in "pin" vs "spin".

## 2. The 10 Basic Vowels

Korean vowels split into **3 shapes**: vertical (ㅣ-based), horizontal (ㅡ-based), and combined.

| Letter | Name | Sound | Mnemonic |
|:---:|:---|:---:|:---|
| ㅏ | 아 (a) | /a/ | like "ah" |
| ㅑ | 야 (ya) | /ja/ | ㅣ + ㅏ |
| ㅓ | 어 (eo) | /ʌ/ | like "uh" but more open |
| ㅕ | 여 (yeo) | /jʌ/ | ㅣ + ㅓ |
| ㅗ | 오 (o) | /o/ | like "oh" |
| ㅛ | 요 (yo) | /jo/ | ㅣ + ㅗ |
| ㅜ | 우 (u) | /u/ | like "oo" |
| ㅠ | 유 (yu) | /ju/ | ㅣ + ㅜ |
| ㅡ | 으 (eu) | /ɯ/ | unique — between "u" and "oo" |
| ㅣ | 이 (i) | /i/ | like "ee" |

## 3. Building Syllables: C+V(+C)

Each Korean syllable block combines **consonant + vowel (+ optional final consonant)**:

```
  가   나   다   라
 ㄱ+ㅏ ㄴ+ㅏ ㄷ+ㅏ ㄹ+ㅏ

  방   밥   산   옷
ㅂ+ㅏ+ㅇ ㅂ+ㅏ+ㅂ ㅅ+ㅏ+ㄴ ㅇ+ㅗ+ㅅ
```

## 4. Five Double Consonants (된소리)

These are **tense consonants** — voice box tightened:

| Letter | Sound | Example |
|:---:|:---:|:---|
| ㄲ | /kː/ | 까다 (kkada) = annoying |
| ㄸ | /tː/ | 떡 (tteok) = rice cake |
| ㅃ | /pː/ | 빠르다 (ppareuda) = fast |
| ㅆ | /sː/ | 싸다 (ssada) = cheap |
| ㅉ | /tʃː/ | 짜다 (jjada) = salty |

## 5. Common Final Consonants (받침)

When a consonant sits at the bottom of a syllable, it's called **받침 (batchim)**. Only 7 sounds are used in pronunciation, even though 14 letters can appear:

| Represented | Actually pronounced as | Examples |
|:---|:---:|:---|
| ㄱ ㄲ ㅋ | ㄱ (k) | 읽**고** = ilko |
| ㄴ | ㄴ (n) | 한국 = han**guk** (but pronounced hanguk) |
| ㄷ ㅅ ㅆ ㅈ ㅊ ㅌ | ㄷ (t) | 있**다** = it**da** |
| ㄹ | ㄹ (l) | 말 = mal |
| ㅁ ㅂ ㅍ | ㅁ (m) | 아름답**다** = areumdap**da** |
| ㅇ | ㅇ (ng) | 방 = bang |
| ㅎ | silent or transferred | 좋**다** = jo**ta** |

> Note: ㄹ is the only 받침 that keeps its unique sound. The rest merge into one of the 7 representative sounds.

## 6. Practical Tips

- ☑️ **Learn the consonants first** — they're easier than vowels
- ☑️ **Practice reading syllables** aloud: 가, 나, 다, 라, 마, 바, 사, 아, 자, 차, 카, 타, 파, 하
- ☑️ **Drill 받침 pronunciation** — many English speakers drop them entirely (a common mistake)
- ☑️ **Use the official names** (기역, 니은...) — it helps in dictation and Hangul games
- ☑️ **한글 is phonetic** — once you learn it, you can read any Korean word out loud
